# Portfolio — Homepage v1 (OpenClaw smoke test)

A production-quality first cut of a personal portfolio homepage.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (subtle animations)
- Jest (component/logic tests where reasonable)

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Scripts

- `npm run lint`
- `npm test`
- `npm run build`
