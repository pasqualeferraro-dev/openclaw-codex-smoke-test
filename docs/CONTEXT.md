# Portfolio Website — Context

This repository contains the Portfolio website for Pasquale Ferraro.

## Goals

- Modern “top company” style homepage
- Bright / energetic / playful (premium, not childish)
- Fully responsive, accessible, and performance-minded
- Subtle motion that respects `prefers-reduced-motion`
- Smooth anchor scrolling with sticky header offset

## Non-goals (v1)

- No CMS
- No analytics
- No complex state management
